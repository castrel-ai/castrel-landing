# Graylog Connector Instructions

The Graylog Connector enables seamless integration with Graylog, allowing Castrel to search and analyze Graylog's logs, streams and fields.

## 📚 Overview

- **Connector Name**: `graylog`
- **Supported API Version**: Graylog REST API
- **Supported Server Versions**: Graylog v6.0.0 and later
- **Official Documentation**: [Graylog Docs](https://go2docs.graylog.org/current/home.htm)

## ⚙️ Configuration

To use the Graylog tools, you need to configure a Connector with the following fields:

| Field | Required | Description |
|-------|----------|-------------|
| `instance_name` | Yes | Instance name to identify this connector instance. |
| `base_url` | Yes | The base URL of your Graylog server API (e.g., `http://graylog:9000/api`). |
| `access_token`| Yes | API Access Token for authentication. |
| `verify_ssl` | No | SSL certificate verification (default: true). |

## 🛠 Available Tools

The connector provides tools for log searching and stream exploration.

- **Log Search**: Search logs using relative time, absolute time, or keywords.
- **Stream Discovery**: List available streams.
- **Field Discovery**: Get available fields.
