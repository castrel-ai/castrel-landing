# Tempo Connector Instructions

The Tempo Connector enables seamless integration with Grafana Tempo, allowing Castrel to query and analyze Tempo's traces, tags and metrics.

## 📚 Overview

- **Connector Name**: `tempo`
- **Supported API Version**: Grafana Tempo HTTP API
- **Supported Server Versions**: Grafana Tempo v2.0.0 and later
- **Official Documentation**: [Grafana Tempo Docs](https://grafana.com/docs/tempo/latest/)

## ⚙️ Configuration

To use the Tempo tools, you need to configure a Connector with the following fields:

| Field | Required | Description |
|-------|----------|-------------|
| `instance_name` | Yes | Instance name to identify this connector instance. |
| `base_url` | Yes | The base URL of your Tempo server (e.g., `http://tempo:3200`). |
| `auth_type`| Yes | Set to `none` or `basic`. |
| `username` | No | Basic Auth username. |
| `password` | No | Basic Auth password. |
| `verify_ssl` | No | SSL certificate verification (default: true). |

## 🛠 Available Tools

The connector provides tools for trace retrieval and analysis.

- **Trace Retrieval**: Get a specific trace by ID.
- **Trace Search**: Search for traces using TraceQL or criteria.
- **Metadata**: List tag names and values to help filter searches.
- **Metrics**: Query metrics derived from traces (RED metrics).
