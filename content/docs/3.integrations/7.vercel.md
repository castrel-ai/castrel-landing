# Vercel MCP Connector Instructions

The Vercel MCP Connector enables seamless integration with Vercel, allowing Castrel to query and analyze Vercel's deployments, projects and runtime logs.

## 📚 Overview

- **Connector Name**: `vercel`
- **Supported API Version**: Vercel REST API
- **Official Documentation**: [Vercel API Docs](https://vercel.com/docs/rest-api)


## 🛠 Available Tools

The connector provides a set of tools for deployment management and analysis.

- **Runtime Logs**: `vercel_get_deployment_runtime_logs` - Retrieve runtime logs (error, warning, info) for a specific deployment.
- **Project Management**: List and inspect Vercel projects.
- **Deployment Management**: List deployments and get deployment details.
