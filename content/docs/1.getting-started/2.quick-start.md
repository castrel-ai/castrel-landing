## Quick Start

Get up and running with Castrel for intelligent operations in just a few steps.

::steps
### Connect an Observability Data Source

First, connect at least one observability data source. Castrel supports major platforms:

- **Prometheus** - Metrics monitoring
- **Elasticsearch** - Log analysis
- **Grafana Loki** - Log aggregation
- **Grafana Tempo** - Distributed tracing

Visit [Integrations](/docs/integrations/prometheus) for complete integration guides.

### Let Castrel Learn Autonomously

Once connected, Castrel will automatically:

1. Explore the data source schema and structure
2. Learn how to query and analyze the data
3. Record learnings to the knowledge base

This process is fully automated with no manual intervention required.

### Start Asking Questions and Troubleshooting

Now you can interact with Castrel using natural language:

- Ask about system health status
- Investigate alert root causes
- Analyze performance issues
- Trace distributed requests

Castrel provides accurate analysis based on your data sources and knowledge base.

### Integrate Slack for the Best Experience

For the best interaction experience, we highly recommend integrating Slack:

- Chat with Castrel directly in Slack
- Receive intelligent alert notifications
- Collaborate with your team on incident investigation

Visit [Slack Integration](/docs/integrations/slack) to complete the setup.
::
